
# 手写数字识别器

这是一个基于Python的手写数字识别程序，提供了多种实现方式，以适应不同的环境和需求。

## 项目结构

```
handwritten_digit_recognizer/
├── README.md                 # 项目说明文档
├── command_line_version.py   # 命令行版本识别器
├── gui_version.py            # 图形界面版本识别器
├── simple_model.py           # 简单的模型实现
└── requirements.txt          # 依赖列表
```

## 功能特点

1. **命令行版本**：不需要图形界面，适合在服务器或无GUI环境下使用
2. **图形界面版本**：提供直观的绘图界面，方便用户输入手写数字
3. **简单模型**：实现了基于模板匹配的识别算法，无需复杂的机器学习库
4. **可扩展**：支持集成更复杂的机器学习模型

## 安装依赖

### 使用pip安装

```bash
pip install -r requirements.txt
```

### 手动安装必要依赖

如果pip安装遇到问题，可以手动安装以下依赖：

```bash
pip install numpy pillow opencv-python
```

## 使用方法

### 1. 图形界面版本

```bash
python gui_version.py
```

- 在白色画布上用鼠标绘制数字
- 点击"识别"按钮获取识别结果
- 点击"清空"按钮重新绘制

### 2. 命令行版本

```bash
python command_line_version.py <image_path>
```

其中 `<image_path>` 是包含手写数字的图片路径。

## 实现原理

### 简单模板匹配

程序使用了基于像素特征的模板匹配算法：
1. 将输入图像调整为统一大小（28x28像素）
2. 二值化处理，将图像转换为黑白
3. 计算输入图像与预定义数字模板的相似度
4. 选择相似度最高的数字作为识别结果

### 可扩展模型

程序预留了扩展接口，可以轻松集成更复杂的机器学习模型，如：
- K近邻算法（KNN）
- 支持向量机（SVM）
- 卷积神经网络（CNN）

## 示例

### 绘制数字示例

1. 运行图形界面版本
2. 在画布上绘制数字
3. 点击识别按钮
4. 查看识别结果

### 命令行示例

```bash
python command_line_version.py test_digit.png
识别结果: 5
```

## 注意事项

1. 绘制数字时尽量居中，保持适当大小
2. 光线充足的环境下识别效果更好
3. 对于复杂的机器学习模型，需要提前训练
4. 某些环境下可能需要额外配置图形界面库

## 技术支持

如果遇到问题，请检查：
1. Python版本是否为3.6及以上
2. 所有依赖是否正确安装
3. 图像路径是否正确
4. 是否有足够的权限运行程序

## 扩展建议

1. 集成TensorFlow或PyTorch等深度学习框架
2. 训练自己的数据集以提高识别准确率
3. 添加更多的预处理步骤，如噪声去除、图像增强等
4. 实现批量识别功能
